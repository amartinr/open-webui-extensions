# open-webui-extensions

Collection of tools, functions, and experiments for [Open WebUI](https://docs.openwebui.com/).

## Structure

```
open-webui-extensions/
├── tools/                        # Workspace Tools (class Tools)
│   └── smart_fetch_url/          #    Fetch URLs with TLS fingerprinting
├── functions/                    # Admin Panel Functions
│   └── filters/                  #   Filters (class Filter)
│       └── deepseek_reasoning/   #      Filters for DeepSeek R1
├── README.md
└── LICENSE
```
